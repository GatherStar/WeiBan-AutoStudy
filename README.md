# WeiBan-AutoStudy

## 介绍
该软件能够帮助学生更快速的完成安全微伴上的微课，更合理的利用时间

## 使用说明
#### 1.使用方式
双击start文件即可启动程序，根据程序提示操作即可
#### 2.软件支持
当前软件仅支持64位架构的处理器运行的Windows系统，后续将更新其他版本支持

下载地址：[Release](https://github.com/lyx20030901/WeiBan-AutoStudy/releases)
